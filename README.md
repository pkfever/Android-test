# Android-test

A Simple Android Mobile Application to display user from github api using MVVM design.


### Technologies & Methodologies which used:

- Koin
- Coroutines
- Clean Architecture
- MVVM Pattern
- LiveData
- Paging3
- Flow
- Moshi
- Coil ( for image loading)
